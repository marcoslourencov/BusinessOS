# Reel — H. Moser & Cie (Remoção como Método)

> **Editoria:** Cases · Cultura
> **Formato:** talking head, tom interpretativo
> **Duração estimada:** 90-120s
> **Status:** pronto para gravar

---

## Ângulo

Existe uma marca de relógio de luxo que apagou o próprio nome do mostrador. E depois removeu o selo "Swiss Made". Cada remoção é uma declaração de soberania. A maioria das marcas adiciona para parecer mais. A Moser remove para provar que já é. Adicionar é fácil — qualquer marca insegura adiciona. Tirar exige saber exatamente quem você é.
